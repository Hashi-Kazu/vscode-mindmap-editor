# Markdown Mind Map Editor

Markdownファイルの見出し構造をインタラクティブなマインドマップとして表示・編集するVS Code拡張機能です。

## 機能

- **マインドマップ表示**: `#`〜`######` 見出しをリアルタイムにマインドマップとして可視化
- **インライン編集**: ノードのダブルクリックでテキストを直接編集
- **ドラッグ&ドロップ**: ノードをドラッグして構造を組み替え
- **ノード追加/削除**: 右クリックメニューから子・兄弟ノードを追加、削除
- **折りたたみ**: ノード単位で子階層を展開/収縮。状態はフロントマターに保存
- **パン&ズーム**: ホイールズーム、ドラッグパン、フィットビュー

## 使い方

1. Markdownファイルをエディタで開く
2. エディタタイトルバーの **⊤** アイコンをクリック、またはショートカット `Ctrl+Shift+M`
3. 横にマインドマップパネルが表示される

## 操作ガイド

| 操作 | 方法 |
|------|------|
| テキスト編集 | ノードをダブルクリック |
| ノード選択 | ノードをシングルクリック |
| 選択ノード削除 | `Delete` キー |
| ノード追加 | ノードを右クリック |
| ドラッグ移動 | ノードをドラッグ |
| 折りたたみ | ノード左端の ▼/▶ をクリック |
| パン | 背景をドラッグ |
| ズーム | マウスホイール |
| フィットビュー | ツールバー ⊡ ボタン、または `F` キー |

## ドラッグ&ドロップの詳細

ノードを別のノードへドロップする際、ドロップ先のどの位置でリリースするかで動作が変わります：

- **上端（25%）**: ドロップ先の**前**（兄弟として）に挿入
- **中央（50%）**: ドロップ先の**子**として追加
- **下端（25%）**: ドロップ先の**後**（兄弟として）に挿入

## 折りたたみ状態の保存形式

折りたたみ状態はMarkdownのフロントマターに保存されます：

```yaml
---
mindmap-collapse:
  - "ルートノード/折りたたまれた節"
  - "ルートノード/別の節/孫"
---
```

## インストール（開発版）

```bash
cd vscode-mindmap-editor
npm install
npm run build
```

生成された `.vsix` ファイルをVS Codeの「拡張機能: VSIXからインストール」でインストールしてください。
